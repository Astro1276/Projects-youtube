## 📂 ¿Cómo descargar los archivos?

Para que sea más fácil, **cada proyecto es un solo archivo**:

1. **Busca el archivo**: El nombre del archivo coincide con el título del video (Ejemplo: `Boton_Neon_Glow.html`).
2. **Todo en uno**: Dentro de ese mismo archivo encontrarás el **HTML, CSS y JS** integrado. Solo copiar y pegar.
3. **Personaliza**: Cambia los colores, las fuentes y hazlo tuyo.
